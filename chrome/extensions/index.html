Chrome Extensions 分享

Posted Jul 7, 2017

整理一下比較常用好用的 chrome extensions (有些Firefox、Safari也有支援)。

Smart TOC

瀏覽網頁時提供 Table of contents 方便瀏覽文章標題。

JSON Formatter

讓 JSON 格式輸出更好看。

Example

<?php

header('Content-Type: application/json;');

$datas = [
    [
        'id' => 1,
        'name' => 'Jack',
        'phone' => '0912345678',
        'habbit' => ['running', 'swimming']
    ]
];

echo json_encode($datas);

Before

After

octotree

瀏覽 GitHub 時，程式碼旁邊樹狀顯示，更方便瀏覽。

ModHeader

修改 HTTP Request Response Headers。

EditThisCookie

新增/編輯/刪除 Cookie。

PostMan

測試開發 API 神器，本來只是 chrome extension，後來變成獨自軟體。

Example

  • 發送 DELETE Request 到 localhost:8000 index.php。
<?php

echo $_SERVER['REQUEST_METHOD'];
  • 結果回傳 DELETE

RequestBin

既然說道API就順便提一下，產生一個 URL 可以收集紀錄 Request 資訊。

Example

  • 首先 Create a RequestBin 會得到一個 URL https://requestbin.com/xxx

  • 用 Postman POST 帶 hello=world 的值過去

  • https://requestb.in/xxx?inspect 查看接收到的訊息

Chrome DevTools Dark Theme

覺得 Chrome DevTools 太亮讓眼睛不舒服嗎？ 馬上簡單換個黑色風格！
