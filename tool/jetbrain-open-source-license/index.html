申請 JetBrains Open Source Licenses

Posted Apr 21, 2020

JetBrains

JetBrains 是一家出產 IDE 產品的公司，幾乎所有語言都有他們的產品，看得懂程式碼，整合了許多方便的工具， 像是會檢查語法、錯字，方便的 unit test 工具，強大的重構功能等。

缺點就是吃很多記憶體，價錢不便宜的，但一分錢一分貨，想增加自己開發效率，不妨可以試試看。

更多 feature 可參考 Top 25 Editing Features of GoLand IDE – Part 1

Open Source License 可以做什麼？

  • 提供 All Products Pack licenses 一年
  • 只能用於非商業開源的專案開發上
  • 只能給專案貢獻者使用
  • more…

如何申請

首先當然要準備一個 open source 的 project，再來到這填表單 https://www.jetbrains.com/shop/eform/opensource

Step 1

問之前是不是有申請過 open source license，第一次就填 No 吧

Step 2

幾個比較需要注意的：

  • Project age: 至少要在過去三個月都有持續的提交
  • Project website: 沒有的話就填 github repository
  • No. of required licenses: 需要的 license 數量，如果是自己一人就填一個吧
  • Project description: 吹噓 描述一下你的 project 吧

這次我申請的專案是 https://github.com/cloudingcity/golab，一個 GitLab CLI 的工具，當初為了熟悉 golang 和輔助自己工作流程寫的，路過的人有空幫我按個星星吧 :D

Step 3

填完剩下的基本資料，確認使用規則就可以送出囉！

結果

大概過了兩週就拿到囉，可喜可賀！

後記

JetBrains 的產品都挺好用的，幫助在我開發上省了大量的時間，在學一個新的語言的時候用 IDE 學也是一個不錯的辦法，某些地方也會提示怎麼寫比較符合 Best Practice， 想要體驗的也有一個月的 Free Trial 可以嘗試，很佛心的公司。
