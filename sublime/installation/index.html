Sublime Text 3 教學

Posted Nov 27, 2016

Sublime Text 3 是一個輕量、簡潔、高效能的編輯器，可以安裝許多套件(Package)， 未註冊版偶爾會跳出詢問購買視窗，功能上與註冊版都一樣，分享一下自己的用法。

安裝

安裝 Sublime Text 3

  • 官網下載對應版本 Sublime Text - Download

安裝 Package Control

  1. Package Control 到此複製安裝代碼

  1. Sublime 開啟 Console ctrl + ` or View > Show Console

  1. 貼上代碼後Enter執行

  1. 開啟 Package Control ctrl + shift + P or Preferences > Package Control

  1. 輸入 install 選擇 Package Control: Install Package

  1. 可開始搜尋安裝套件

套件

套件功能
convertToUTF8轉換邊碼，正常顯示中文
Color Highlighter顯示色碼顏色
AutoFileName自動抓取檔案路徑
Emmet快速建立 html code，Emmet cheat sheet
BracketHighlighter括號高亮顏色顯示
PackageResourceViewer查看編輯套件
Tomorrow Color SchemesTomorrow 風格高亮程式碼
Seti_UISeti 風格介面

設定

  • Sublime 設定是以 JSON 格式去做存取，Preferences > Setting，右邊可以自定義設定

名稱 | 值 | 作用 — | — font_size | number | 字體大小 draw_white_space | “all” / “none” | 顯示空白 show_encoding | true / false | 右下角顯示檔案編碼 tab_size | number | tab 代表幾個空白 trim_trailing_white_space_on_save | true / false | 儲存後去檔案空白 word_wrap | true / false | 程式碼換行

熱鍵

熱鍵功能
ctrl+H搜尋取代
ctrl+G搜尋行數
ctrl+R搜尋函式類別
ctrl+Y回覆上一步
ctrl+L選取整行
ctrl+shift+V按照格式貼上
ctrl+Enter跳到下一行縮排開始
ctrl+/註解/移除
ctrl+shift+/區域註解/移除
ctrl+N開啟新分頁
ctrl+W關閉當前分頁
ctrl+shift+T還原最後關閉分頁
ctrl+F2設定書籤
F2到下一個書籤
alt+shift+1~4垂直分割視窗(1~4)
ctrl+K, ctrl+J展開所有程式碼
ctrl+K, ctrl+1~4摺疊所有程式碼1~4層數
ctrl+K, ctrl+B切換側邊欄位
ctrl+shift+P開啟命令列
