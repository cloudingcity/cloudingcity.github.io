Posted Nov 19, 2017

PHP 陣列最後一個值的結尾逗號要加還是不加?

加不加都可以，但加了有兩個好處:

  1. 方便未來要新增值進去時，上一筆結尾不需再加逗號

  2. 在版本控制中差異顯示的會是改一行，而不是兩行

if else 技巧

if else 有時候會造成程式越寫越多層，產生了大家所說的波動拳程式碼，運用一些小技巧讓處理條件更清楚明確。
