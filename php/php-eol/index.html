PHP_EOL 常數對應各種平台換行符號

Posted Jan 16, 2017

PHP_EOL 常數對應各種平台換行符號，可以讓程式碼適應各種環境。

Table

作業系統縮寫ASCIICode
WindowsCR+LF13+10\r\n
UnixLF10\n
pre-OS XCR13\r
OS XLF10\n

Test

<?php

/**
 * 同等於
 * echo "\r\n"; (Windows)
 * echo "\n"; (Unix)
 * echo "\n"; (OS X)
 */
echo PHP_EOL;
